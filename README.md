# Sathwik Neelisetty — Professional Portfolio v2

A high-performance, premium portfolio website designed with a **Deen Dayal** inspired aesthetic. This project showcases data-driven backend expertise and modern frontend design principles.

![Portfolio Preview](https://images.unsplash.com/photo-1517694712202-14dd9538aa97?auto=format&fit=crop&q=80&w=1200&h=600)

## ✨ Key Features

- **Premium UI/UX**: Glassmorphic interfaces, glowing hover effects, and smooth scroll-reveal animations.
- **Dynamic Theme Engine**: Seamlessly toggle between deep dark mode and a crisp, modern light theme.
- **Interactive API Sandbox**: Live data fetching integration demonstrating backend-to-frontend synchronization.
- **Responsive Architecture**: Fully optimized for mobile, tablet, and desktop viewing.
- **Formspree Integration**: Operational contact form for lead generation and professional outreach.

## 🛠️ Tech Stack

- **Frontend**: Vanilla HTML5, CSS3 (Modern Flex/Grid), JavaScript (ES6+)
- **Icons**: Tabler Icons
- **APIs**: Integrated Weather and GitHub stats
- **Deployment**: Local testing via Python HTTP server, production-ready for GitHub Pages / Netlify.

## 📁 Project Structure

- `index.html`: Main semantic structure and layout.
- `styles.css`: Custom premium design system and theme overrides.
- `script.js`: Interactive logic, theme management, and API integrations.
- `Sathwik_CV_v5 (1).docx`: Integrated professional resume.

## 🚀 Local Development

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Sathvik1696/Portifolio.git
   ```

2. **Run a local server**:
   You can use Python's built-in server:
   ```bash
   python -m http.server 8080
   ```

3. **Open in browser**:
   Navigate to `http://localhost:8080` to view the site.
